Popr

Popr is a social media patform for movie lovers. Users can join movie events created by their friends on the platform. In addition, users can populate their profiles with reviews and ratings of movies they've seen, as well as a wishlist of movies they intend to watch.

The website makes use of the Rotten Tomatoes and Google Maps APIs.

This project was created as a final project for the immersive web development program at Bitmaker Labs. Click for the GitHub repo.

Earned Value Management Tool

Earned Value Management is a method for project management. The EVM Tool allows for efficient monitoring of budget and schedule performance against baseline figures. 'Actual' figures from the progress periods are compared with the 'Planned' figures and performance metrics are provided. The technique is widely used in the construction industry, among others.

Residency/Citizenship Calculator

The Residency and Citizenship Calculator is a log that keeps track of a user's travel history. Based on the number of days absent from Canada, the app provides residents with the ideal date to apply for renewal of residency or for citizenship after having completed the residency requirements.

Click for the GitHub repo. Please note this app is not official and provides no guarantees.
